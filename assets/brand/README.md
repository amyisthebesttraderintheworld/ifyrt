# IFYRT Brand Assets

## Primary asset

- `ifyrt-logo-primary.png`: current IFYRT logo featuring the upward bar mark and wordmark.

## Usage

- Use this asset as the default logo for repository docs, future dashboard work, and launch materials unless a newer approved revision replaces it.
- Keep the asset in `assets/brand/` so future frontend and marketing surfaces have one canonical path to pull from.
